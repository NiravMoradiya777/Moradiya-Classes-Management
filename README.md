## How To Use This Software
[![Moradiya Classes Management Demo](https://i.ytimg.com/vi/JMIDds1eQ_g/hqdefault.jpg?sqp=-oaymwEcCNACELwBSFXyq4qpAw4IARUAAIhCGAFwAcABBg==&rs=AOn4CLAagy9XPxK0qR046E8F8I2lRpB9Fg)](https://www.youtube.com/watch?v=JMIDds1eQ_g)

### Admin Panel

1. **Manage Students:**
   - Add, edit, or delete student records.
   - Change student fees based on standards.

2. **Manage Student Fees:**
   - Track and update student fee records.
   - Generate PDF receipts for fee payments.

3. **Manage Student Marksheet:**
   - View and update student marksheets.
   - Add exam dates and manage exam-related information.

4. **Student Fees Payment:**
   - Record and manage student fee payments.
   - Automatically generate PDF receipts.

5. **Manage Teachers:**
   - Add, edit, or remove teacher profiles.
   - Track teacher payments and information.

6. **Manage Teacher Payments:**
   - Record and manage teacher salary payments.

7. **Change Course for Standards:**
   - Modify the course details for each standard.

8. **Create Exam Dates:**
   - Set and manage exam dates.

### Teacher Panel

1. **Manage Students:**
   - View student information.

2. **Take Student Fees:**
   - Record student fee payments.

3. **Add Exam:**
   - Add new exams and relevant details.

4. **Add Student Marks:**
   - Input and update student marks.
